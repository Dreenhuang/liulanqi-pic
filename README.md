# 浏览器看图片 - Chrome 图片浏览扩展

一个强大的 Chrome 浏览器扩展，让你在网页上像本地图片浏览器一样方便地浏览图片，支持自动翻页、图片预加载、全屏浏览、一键下载等功能。

![版本](https://img.shields.io/badge/version-3.0.0-blue.svg)
![许可证](https://img.shields.io/badge/license-MIT-green.svg)

## 功能特性

### 核心功能
- **一键浏览**：点击网页上的任意图片，自动提取页面中所有符合条件的图片
- **左右翻页**：使用键盘左右方向键或点击左右按钮切换图片
- **自动加载下一页**：浏览到当前页最后一张图片时，自动加载下一页的图片
- **智能过滤**：自动过滤小图片，支持自定义过滤尺寸
- **图片预加载**：自动预加载后续 2-3 页的图片，提高浏览流畅度
- **一键下载**：支持一键下载所有浏览的图片

### 浏览模式
- **全屏模式**：打开图片查看器后自动进入全屏，图片占满整个屏幕
- **悬浮按钮**：页面右下角显示可拖拽的紫色悬浮按钮，点击即可打开查看器
- **插件按钮**：点击浏览器工具栏的扩展图标，同样可以打开图片查看器

### 图片操作
- **缩放功能**：
  - 按住 Ctrl + 滚轮进行缩放
  - 点击缩放按钮（+ / -）进行缩放
  - 点击适应屏幕按钮（⊡）恢复原始大小
  - 点击图片本身可在 100% 和 200% 之间切换
- **拖拽查看**：放大后可以拖拽查看图片不同区域
- **一键下载**：左下角下载按钮，一键保存所有图片

### 缓存状态
- 全屏模式下左上角显示缓存状态
- 显示已缓存的页数和图片数量
- 鼠标悬停可查看每页的详细缓存信息
- 实时更新缓存进度

### 自定义设置
- **图片过滤**：可自定义最小图片宽度和高度（默认 600×600 像素）
- **下载路径**：可设置默认下载路径（默认 D:\PIC）

### 快捷键
| 按键 | 功能 |
|------|------|
| ← (左方向键) | 上一张图片 |
| → (右方向键) | 下一张图片 |
| ESC | 退出图片查看器 |
| Ctrl + 滚轮 | 缩放图片 |
| Ctrl + 0 | 打开/关闭图片查看器 |

## 安装方法

### 方法一：Chrome 网上应用店（推荐）
1. 访问 Chrome 网上应用店
2. 搜索 "浏览器看图片"
3. 点击 "添加至 Chrome"

### 方法二：开发者模式安装
1. 下载本仓库代码并解压
2. 打开 Chrome 浏览器，输入 `chrome://extensions/` 进入扩展管理页面
3. 打开右上角的 "开发者模式"
4. 点击 "加载已解压的扩展程序"
5. 选择解压后的文件夹

## 使用方法

### 基本使用
1. 访问包含图片的网页（如图片分享网站）
2. 点击页面上的任意图片，或点击右下角的紫色悬浮按钮，或点击浏览器工具栏的扩展图标
3. 图片查看器将以全屏模式打开，显示页面中所有符合条件的图片
4. 使用左右方向键或点击左右按钮浏览图片
5. 按 ESC 键退出查看器

### 悬浮按钮
- 页面右下角会显示一个紫色圆形悬浮按钮
- 按钮可以拖拽到任意位置
- 点击按钮打开图片查看器

### 图片筛选
扩展会自动过滤以下图片：
- 尺寸小于设定值的图片（默认 600×600 像素，可在设置中调整）
- 包含 icon、logo、avatar、button、loading、thumb、small、banner、ad、gif 等关键词的图片

### 自动翻页
- 浏览到当前页最后一张图片时，会自动加载下一页
- 支持常见的分页 URL 格式，如 `page_1.html`、`page_2.html` 等
- 预加载后续页面，确保浏览流畅

### 下载图片
1. 打开图片查看器
2. 点击左下角的 "⬇️ 下载全部" 按钮
3. 确认下载后，所有图片将批量下载到浏览器默认下载文件夹
4. 下载完成后会显示成功/失败数量

### 自定义设置
1. 点击扩展图标，选择 "⚙️ 设置"
2. 或右键点击扩展图标，选择 "选项"
3. 在设置页面中：
   - 调整图片过滤的最小宽度和高度
   - 设置默认下载路径（提示信息用）
4. 点击 "保存设置"

## 技术实现

### 文件结构
```
├── manifest.json      # 扩展配置文件
├── background.js      # 后台服务脚本
├── content.js         # 内容脚本（核心逻辑）
├── popup.html         # 弹出窗口 HTML
├── popup.js           # 弹出窗口脚本
├── options.html       # 设置页面 HTML
├── options.js         # 设置页面脚本
├── viewer.css         # 查看器样式（备用）
└── icon16.svg         # 扩展图标
```

### 核心技术
- **Manifest V3**：使用 Chrome 扩展最新版本规范
- **Content Script**：注入页面提取图片和处理用户交互
- **Background Service Worker**：处理扩展生命周期事件
- **动态 URL 解析**：智能识别分页 URL 格式
- **图片预加载**：提前加载后续页面图片到缓存
- **Chrome Storage API**：保存用户设置

### 分页识别逻辑
支持多种分页 URL 格式：
- `xxx_1.html` → `xxx_2.html` → `xxx_3.html`
- `xxx.html` → `xxx_2.html` → `xxx_3.html`
- 自动识别页码并生成下一页 URL

## 注意事项

1. **刷新页面**：安装或更新扩展后，需要刷新网页才能使用
2. **权限申请**：扩展需要访问网页内容、存储和下载权限，请在使用时允许
3. **兼容性**：仅支持 Chromium 内核的浏览器（Chrome、Edge 等）
4. **图片加载**：部分网站的图片可能有防盗链机制，导致无法加载或下载
5. **下载路径**：由于浏览器安全限制，实际下载路径以浏览器设置为准。建议在 Chrome 设置中配置下载位置

## 常见问题

**Q: 为什么有些图片没有显示？**  
A: 扩展默认过滤小于 600×600 像素的图片。您可以在设置中调整过滤条件。

**Q: 下载的图片保存在哪里？**  
A: 图片会保存到浏览器的默认下载文件夹。您可以在 Chrome 设置中更改下载位置。

**Q: 悬浮按钮不见了怎么办？**  
A: 刷新页面后，按钮会自动出现。如果仍不显示，请检查扩展是否已启用。

**Q: 如何修改图片过滤尺寸？**  
A: 点击扩展图标 → 设置 → 修改最小宽度和高度 → 保存设置。
